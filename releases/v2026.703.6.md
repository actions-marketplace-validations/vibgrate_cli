# Vibgrate CLI 2026.703.6

_Released 2026-07-03_

This release of the Vibgrate CLI addresses memory issues encountered during baseline and maximum-privacy scans on large codebases. The changes ensure that these scans complete successfully without exhausting system resources.

## What changed

### Changed

- These scans now intentionally skip the local code map, preventing memory exhaustion during the scan process.

### Fixed

- Baseline and maximum-privacy scans on very large codebases no longer run out of memory.
- Large baseline scans now finish and publish reliably, avoiding previous aborts that affected the publish step.
