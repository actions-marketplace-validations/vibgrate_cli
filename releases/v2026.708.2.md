# Vibgrate CLI 2026.708.2

_Released 2026-07-08_

This release of the Vibgrate CLI includes a fix for the `vg update` command, improving its functionality when used in a pnpm workspace. Users will now receive an interactive prompt to install with the workspace-root flag, enhancing usability in CI and scripts.

## What changed

### Fixed

- `vg update` now recognizes when run at a pnpm workspace root and offers to install with the workspace-root flag instead of failing.
